##### [爬楼梯](https://leetcode.com/problems/climbing-stairs) 

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
示例 2：

输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶



```c

//第i阶的爬法数量 = 第i-1阶的爬法数量 + 第i-2阶的爬法数量

class Solution {
public:
    int climbStairs(int n) {
        std::vector<int> dp(n+3,0);
        dp[1] = 1;
        dp[2] = 2;
        for (int i = 3; i < n;i++){
            dp[i] = dp[i-1] + dp[i-2];
        }
        return dp[n];
    }
};

```
